# ⚡ TensorBench

**Benchmark your AI hardware. Get real metrics & smart upgrade advice.**

TensorBench is a lightweight, privacy-first tool that measures how well your PC/server handles local AI workloads. Unlike theoretical specs, it runs actual inference tests and tells you exactly what to upgrade for the best performance-per-dollar.

##  Key Features
- 🖥️ **Real Hardware Detection**: CPU, GPU, VRAM, RAM, Disk, OS
- 🤖 **Live AI Inference Tests**: TTFT, Tokens/sec, VRAM/RAM usage, thermal throttling
-  **Bottleneck Analyzer**: Identifies what's holding you back (GPU, RAM, CPU, or Storage)
- 💰 **Upgrade ROI Calculator**: Compares cost vs. performance gain for different upgrades
- 🔒 **Privacy First**: All tests run locally. No data leaves your machine without consent.
